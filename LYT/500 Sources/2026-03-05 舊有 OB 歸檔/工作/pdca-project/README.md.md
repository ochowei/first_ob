# 操作指引（PDCA 對應）

- **Plan**：只在重大變更時更新 `goals` 與 `roadmap`；保持精簡與明確。
    
- **Do**：新增/更新 `todos`；日誌中引用 `T-*`、`M-*`；完成就把 `status: done`。
    
- **Check**：每週檢視 `roadmap` 的 `status` 與 `risk`；把阻塞原因寫入 `notes`。
    
- **Act**：若目標要調整，更新 `goals`/`roadmap`，並在 `.pdca-project/04_Act/adjustments_YYYY-MM.md` 紀錄決策脈絡（可引用此檔的 ID）。
    

> 小技巧：每日/每週例行動作以「**只改 status/notes/estimate/due**」為主，避免結構漂移。
