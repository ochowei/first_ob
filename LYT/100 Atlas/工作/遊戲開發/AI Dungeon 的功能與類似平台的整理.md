# 🔍 AI Dungeon 功能總覽

## 🎮 基本玩法

| 功能                         | 說明                     |
| -------------------------- | ---------------------- |
| **Do / Say / Story / See** | 以自然語言描述行動、對話、敘述或視覺內容   |
| **自由敘事**                   | 玩家可自創世界觀、角色與劇情，無固定腳本   |
| **多人模式**                   | 可與他人協作冒險，輪流輸入敘事與選擇     |
| **主題類型**                   | 包含奇幻、科幻、末日、現代、歷史、成人等風格 |

## 🆓 免費版限制

| 限制項目          | 說明                        |
| ------------- | ------------------------- |
| **上下文長度**     | 約 2,000 tokens（約 1,500 字） |
| **AI 模型**     | 使用基礎模型（非 GPT-3 / Dragon）  |
| **記憶容量**      | 最多 25 條記憶                 |
| **圖像生成與世界卡片** | 僅限部分功能或不可使用               |

## 💎 付費功能（高級會員）

|功能|說明|
|---|---|
|**高階 AI 模型**|使用 Dragon 或 Mistral 模型，文字品質更高|
|**上下文延伸**|可達 4,000～8,000 tokens，更長篇、更連貫|
|**記憶系統**|自訂角色、設定、背景，支援長期故事發展|
|**Story Cards**|可視化管理角色、地點、物品等世界觀資料|
|**AI Art 生成**|自動生成插圖（每日限制張數）|
|**NSFW 模式**|解鎖成人題材與更寬鬆的敘事主題|

# 🧩 類似平台比較

|平台名稱|特色與用途|適合對象|網站連結|
|---|---|---|---|
|**NovelAI**|強大敘事能力、支持日系風格圖像、支援記憶與風格模板|輕小說作者、創作小說者|[novelai.net](https://novelai.net/)|
|**Character.AI**|角色互動為主，建立虛擬 AI 角色對話|想和 AI 聊天、模擬角色互動的人|[beta.character.ai](https://beta.character.ai/)|
|**Replikant**|原 Replika，主打情感 AI、亦可劇情模擬|虛擬伴侶、戀愛互動玩家|[replikant.ai](https://replikant.ai/)|
|**TextSynth**|提供開源 AI 模型（如 GPT-J），偏向技術用戶|喜歡控制 prompt 的創作者|[textsynth.com](https://textsynth.com/)|
|**Inklewriter**|非 AI，但支援分支式互動小說設計|腳本設計者、遊戲設計師|[inklewriter](https://www.inklestudios.com/inklewriter/)|
