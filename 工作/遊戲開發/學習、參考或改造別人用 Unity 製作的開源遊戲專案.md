## 🌍 主流開源遊戲資源平台（支援 Unity）

### 1. GitHub 搜尋（最推薦）

🔍 搜尋關鍵字建議：

- unity visual novel
- unity open source game
- unity narrative game
- unity 2D game

🧠 實**用技巧：**

在搜尋時加上 stars:>100 可以找到較多人關注的專案

範例搜尋語法：

`unity visual novel stars:>100`

---

### 2. itch.io - Open Source 專區

這是獨立開發者的天堂，很多開源遊戲會同時提供 **可玩的版本 + GitHub 專案連結**

✅ 優點：

- 有完成度的 demo 可玩
- 有些專案支援 WebGL 直接線上試玩
- 介面清楚、偏向小型創意專案，適合學習

---

### 3. Unity Asset Store - 免費模板

雖然不是開源遊戲，但你可以找到：

- 免費的 **視覺小說框架**
- 模擬互動系統模板（例如：對話系統、任務系統）

🔍 搜尋關鍵字： visual novel, dialogue system, narrative toolkit

⚠️ 注意 Asset Store 大多是商用授權（非開源），**可用但不可改作/釋出**

---

### 4. Awesome-Unity 開源整理資源

GitHub 上一份非常完整的 Unity 開源專案與資源清單，包含：

- 教學專案
- 完整遊戲
- 工具模組（對話系統、存檔系統等）
